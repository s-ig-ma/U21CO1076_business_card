#  Business Card Android App

##  Project Description

This project is a simple **Android Business Card application** built using **Kotlin** and **Jetpack Compose** in Android Studio.

The application displays a digital business card containing personal and contact information in a clean and modern user interface.

The purpose of this project is to demonstrate how **Jetpack Compose composables** can be used to build structured and reusable UI components in Android applications.

---

##  Features

* Built with **Kotlin**
* Uses **Jetpack Compose** for UI development
* Clean **brown, light brown, and white themed interface**
* Profile section with image and personal information
* Contact section with icons for phone, social media, and email
* Modular UI built with reusable composables

---

## 🛠 Technologies Used

* Kotlin
* Jetpack Compose
* Android Studio
* Material Design Icons

## How to Clone and Run the Project

### 1)Clone the Repository

Open a terminal and run:

```
git clone https://github.com/s-ig-ma/U21CO1076_business_card.git
```

---

### 2) Open the Project

1. Open **Android Studio**
2. Click **Open**
3. Navigate to the cloned project folder
4. Select the folder `U21CO1076_business_card`

---

### 3) Sync Gradle

Android Studio will automatically start **Gradle Sync**.
Wait until the sync completes successfully.

---

### 4) Run the Application

1. Connect an **Android device** or start an **Android Emulator**
2. Click the **Run ▶ button** in Android Studio
3. The Business Card application will launch on the device/emulator
